# Splitwise - AI Enabled SettleUp

Splitwise is a modern, AI-powered Android application designed to simplify shared expense tracking and settlement. Built with Jetpack Compose and focusing on user experience, it allows users to easily manage group expenses, scan receipts using AI, and settle debts efficiently.

*   **Smart Receipt Scanning**: AI-powered receipt scanning (OCR) to automatically extract purchase details and split expenses without manual entry.
*   **Group Management**: Create and manage multiple groups for different occasions (trips, housemates, events).
*   **Expense Tracking**: detailed logs of all shared expenses, including categories and payer information.
*   **Settlements**: Algorithm-optimized settlement plans to minimize the number of transactions needed to pay everyone back.
*   **Multi-Currency Support**:  Handle expenses in various currencies.
*   **Interactive UI**: A beautiful, fluid user interface built with Jetpack Compose, featuring dark mode and smooth transitions.
*   **Secure Authentication**: User sign-up and login functionality.
